# Guía de Instalación y Uso — Fácil Diapos

¡Bienvenido a **Fácil Diapos**! Esta carpeta contiene los archivos de distribución listos para ejecutar la aplicación de creación de diapositivas en tu computadora.

---

## 📦 Contenido de esta Carpeta

* **`Fácil Diapos Setup 0.0.0.exe`**: Instalador oficial para computadoras con **Windows**.
* **`manual_macos.md`**: Guía paso a paso en formato Markdown detallando cómo instalar, ejecutar y evadir las alertas de seguridad de Gatekeeper en **macOS**.
* **`README.md`**: Este archivo de instrucciones de inicio rápido.

---

## 🚀 Cómo Empezar

### 💻 En Windows (Instalación Rápida)
1. Haz doble clic sobre el archivo **`Fácil Diapos Setup 0.0.0.exe`**.
2. El programa de instalación se iniciará de forma automática.
3. Tras finalizar, creará un acceso directo en tu Escritorio y abrirá la aplicación **Fácil Diapos** lista para usarse offline.

### 🍎 En macOS (Apple Mac)
1. La aplicación puede ser compilada directamente en una Mac o descargada desde los compilados automáticos de GitHub Actions.
2. Debido a las políticas de seguridad de Apple para aplicaciones no firmadas con cuenta de pago, el sistema de seguridad de macOS (Gatekeeper) bloqueará el inicio por defecto.
3. **Consulta el manual adjunto `manual_macos.md`** donde se explica detalladamente cómo saltar este bloqueo en 2 segundos haciendo *Clic Derecho -> Abrir* para configurar una excepción permanente.

---

## 🌐 Alternativa de Uso en el Navegador (Sin Instalar)
Si prefieres no instalar ninguna aplicación de escritorio o te encuentras en un entorno corporativo o académico con restricciones de administrador:
1. Dirígete a la carpeta `dist/` de este proyecto.
2. Haz doble clic sobre el archivo `index.html`.
3. Se abrirá la aplicación de forma instantánea en tu navegador web de preferencia (Google Chrome, Safari, Microsoft Edge, Firefox, etc.) sin necesidad de estar conectado a internet.
4. Contará con el 100% de la funcionalidad de edición, importación de archivos CSV, exportación a PowerPoint (.pptx) y descargas a PDF.
